API

Respuesta de frecuencia "detallada" y descripción:

API siempre predominó en Baterias, guitarras y todo aquello a lo que se le quiera respetar o agregar transientes. Ha sido una referencia a nivel mundial en cuanto a audio profesional se refiere, de hecho, fue la respuesta americana al tan afamado sonido inglés, Neve y SSL, pero nunca habia llegado tanto al usuario medio. Hoy en día todo el mundo habla de previos API. Es un sonido muy vivo, agresivo y con pegada (punchy); se podría decir que tirando a oscurito y con graves definidos, que realza los armónicos de las frecuencias medias, especialmente en guitarras, así como unos agudos suaves y dulces. Es muy común usar previos API para grabar instumentos de percusión, ya que son muy "rápidos" y tienen mucha pegada. Los bombos, cajas y timbales de la batería suenan espectaculares con estos previos, con mucho ataque y definición, un sonido muy "gordo". Otra especialidad del sonido API son las guitarras, ya que tienen un color muy bonito en todo el rango de frecuencias medias. Las guitarras acústicas suenan muy grandes y con mucho cuerpo gracias a estos previos.

Opinión personal: Comparado con todos en general, imprimió un tono un tanto opaco a sus frecuencias medias y altas, parecía carecer de volumen general, cabe destacar que el Plugin usado es el creado por la marca Lindell powered by Alliance, quizás con otra marca no pasaría eso, todo puede depender del modelo de API que fue replicado. Visualicé que sus picos de decibeles eran inferiores a los de otras consolas, por lo que agregué un Dbs para emparejar el Peack con el resto, igualmente sentí que sus medios y agudos no tenían tanta presencia como otros. Destaco sus graves y como respeta las transientes de la guitarra eléctrica.

Usa su Comp. API-2500

Elije attack rápido y release rápido/medio para instrumentos con volumen muy disparejo, reduciendo 2 o 3 decibeles, luego con el output añade la cantidad de dbs perdidas y lograras un sonido mas parejo, uniforme y con pegada.

Usa el mismo en baterias

Con attack lento y release rápido lograrás un efecto pegamento reduciendo 2 o 3 decibeles, luego con el output añade la cantidad de dbs perdidos y lograras un sonido mas homogéneo, presente y uniforme en su volumen final.

Úsalo como pre-amp

Insértalo sin modificar sus knobs para simular el uso de pre-amps que aportan su textura, luego puedes modificar su gain de salida y tendrás un sonido con una capa de distorcion agradable al oido del oyente sin dañar oidos.
